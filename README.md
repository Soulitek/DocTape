# יומן גיבוי קלטות אופסייט

אפליקציית ווב מקומית לניהול מעקב אחר מחזור החיים של קלטות גיבוי LTO עבור MSP.

## תכונות

- ✅ ניהול מלא של רשומות (CRUD) - הוספה, עריכה, מחיקה
- ✅ חיפוש וסינון בזמן אמת
- ✅ מיון לפי עמודות
- ✅ שמירה אוטומטית ב-LocalStorage
- ✅ ייצוא ל-CSV ול-JSON
- ✅ ייבוא מ-JSON
- ✅ ממשק בעברית עם תמיכה ב-RTL
- ✅ תמיכה במובייל (Responsive)
- ✅ עובד ללא חיבור לאינטרנט

## שימוש

1. פתח את הקובץ `tape-log.html` בדפדפן (כפול קליק)
2. האפליקציה תעבוד מיידית ללא צורך בהתקנות

## שדות הטופס

### שדות חובה:
- **ברקוד קלטת** - מזהה ייחודי של הקלטת
- **תאריך גיבוי אחרון** - תאריך הגיבוי האחרון

### שדות נוספים:
- **מערכות מגובות** - רשימת המערכות שהוגנו
- **תאריך ייצוא** - תאריך הייצוא לאופסייט
- **שעת ייצוא** - שעת הייצוא
- **מייצא** - שם האחראי על הייצוא (חובה אם תאריך ייצוא מוזן)
- **יעד אופסייט** - מיקום האחסון (חובה אם תאריך ייצוא מוזן)
- **תאריך החזרה** - תאריך החזרה מהאופסייט
- **הערות** - הערות נוספות

## פורמט תאריכים

- תאריכים מוצגים בפורמט: **DD/MM/YYYY**
- שעות בפורמט: **24 שעות** (HH:MM)

## שמירת נתונים

הנתונים נשמרים אוטומטית ב-LocalStorage של הדפדפן. 
לשמירה חיצונית, השתמש בפונקציות ייצוא CSV/JSON.

## דרישות

- דפדפן מודרני (Chrome, Firefox, Edge, Safari)
- אין צורך בחיבור לאינטרנט
- אין צורך בהתקנות

## טכנולוגיות

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage API
