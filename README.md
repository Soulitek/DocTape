# יומן גיבוי קלטות אופסייט

אפליקציית ווב מקומית לניהול מעקב אחר מחזור החיים של קלטות גיבוי LTO עבור MSP.

## תכונות

- ✅ ניהול מלא של רשומות (CRUD) - הוספה, עריכה, מחיקה
- ✅ חיפוש וסינון בזמן אמת
- ✅ מיון לפי עמודות
- ✅ שמירה אוטומטית ב-LocalStorage
- ✅ ייצוא ל-CSV ול-JSON
- ✅ ייבוא מ-JSON
- ✅ ממשק בעברית עם תמיכה ב-RTL
- ✅ תמיכה במובייל (Responsive)
- ✅ עובד ללא חיבור לאינטרנט

## שימוש

1. פתח את הקובץ `tape-log.html` בדפדפן (כפול קליק)
2. האפליקציה תעבוד מיידית ללא צורך בהתקנות

## שדות הטופס

### שדות חובה:
- **ברקוד קלטת** - מזהה ייחודי של הקלטת
- **תאריך גיבוי אחרון** - תאריך הגיבוי האחרון

### שדות נוספים:
- **מערכות מגובות** - רשימת המערכות שהוגנו
- **תאריך ייצוא** - תאריך הייצוא לאופסייט
- **שעת ייצוא** - שעת הייצוא
- **מייצא** - שם האחראי על הייצוא (חובה אם תאריך ייצוא מוזן)
- **יעד אופסייט** - מיקום האחסון (חובה אם תאריך ייצוא מוזן)
- **תאריך החזרה** - תאריך החזרה מהאופסייט
- **הערות** - הערות נוספות

## שמירת נתונים

### שמירה נוכחית
הנתונים נשמרים אוטומטית ב-LocalStorage של הדפדפן. 
לשמירה חיצונית, השתמש בפונקציות ייצוא CSV/JSON.

### אופציות שמירה נוספות

#### 1. שמירה ידנית (מומלץ להתחלה) ⭐
- שימוש בפונקציות ייצוא CSV/JSON הקיימות
- שמירת קבצים ב-Google Drive / OneDrive / Dropbox
- **יתרונות:** כבר קיים, שליטה מלאה, גיבוי פשוט

#### 2. Firebase Realtime Database ⭐ מומלץ
- סנכרון אוטומטי בזמן אמת
- חינמי עד 1GB
- **יתרונות:** קל להטמעה, סנכרון אוטומטי, אבטחה מובנית

#### 3. Supabase ⭐ מומלץ
- PostgreSQL בענן + API
- חינמי עד 500MB
- **יתרונות:** SQL database, גמיש, API אוטומטי

#### 4. שרת עצמי
- API פשוט על שרת משלך
- **יתרונות:** שליטה מלאה, אין מגבלות

#### 5. GitHub Gist
- שמירת JSON ב-GitHub Gist
- **יתרונות:** חינמי, גרסה/היסטוריה

#### 6. AWS S3 / Azure Blob Storage
- אחסון קבצים בענן
- **יתרונות:** אמין ומדרגי

> 📖 **לפרטים נוספים:** ראה [DATA_STORAGE_OPTIONS.md](DATA_STORAGE_OPTIONS.md) למידע מפורט על כל אופציה
